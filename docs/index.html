Video Study AI

Transforme vídeos em provas de estudo interativas — extração, transcrição e geração de questões automática.

Demo & Código

O código-fonte está disponível no GitHub. Para executar localmente, clone o repositório e siga o README.

Ver no GitHub Release v0.1.0

Como usar

  1. Cole a URL do vídeo (YouTube/TikTok/Instagram)
  2. Escolha o número de questões
  3. Clique em "Processar Vídeo" no app Gradio

Observações

  • Melhor com áudio claro em PT/EN
  • Recomenda-se Python 3.8+ e ffmpeg instalado
